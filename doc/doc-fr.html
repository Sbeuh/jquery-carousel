Fonctionnement

Le carrousel est conçu pour être totalement accessible lorsque Javascript est désactivé.
Voici un exemple de structure HTML sur laquelle pourra se baser le script : 

<div id="my-carousel">
	<ul>
		<li>item 1</li>
		<li>item 2</li>
		<li>item 3</li>
		<li>item 4</li>
		<li>item 5</li>
		<li>item 6</li>
	</ul>
</div>

Et le script permettant de l'initialiser :

<script type="text/javascript">$("#my-carousel").carousel();</script>

Il est ainsi possible de définir un carrousel sur une simple liste.
Cela peut etre appliqué sur différentes structures. Le seul prérequis est d'avoir deux conteneurs à l'intérieur desquels sera contenue une suite d'éléments enfants.
(Dans l'exemple ci-dessus, l'élément <ul> peut etre remplacé par un élément <div> et les éléments <li> par des éléments <p>, etc.)

Utilisation

Il est possible de passer des options à la méthode jquery.carousel, sous la forme d'un tableau associatif. Ces paramètres sont détaillés à la suite.

direction

Type attendu : string

Valeur par défaut : "horizontal"

Valeurs possibles : "horizontal", "vertical"

Le paramètre direction permet de définir la direction du carrousel (horizontal ou vertical).

Exemple :

// Carrousel vertical
$("div").carousel( { direction: "vertical" } );

paginationPosition

Type attendu : string

Valeur par défaut : "inside"

Valeurs possibles : "inside", "outside"

Dépend de : pagination défini à true

Le paramètre paginationPosition permet de définir la position de la pagination par rapport au conteneur global du carrousel.

Exemple :

// Carrousel avec pagination
$("div").carousel( {
	pagination: true,
	paginationPosition: "outside"
} );

nextBtn / prevBtn

Type attendu : string

Valeurs par défaut : "<span role="button">Next</span>" et "<span role="button">Previous</span>"

Les paramètres nextBtn et prevBtn permettent de définir le code HTML des bouton "Next" et "Previous", respectivement.

Exemple :

// Carrousel avec des images pour les boutons de contrôle
$("div").carousel( { 
	nextBtn: '<img src="images/buttons/next.png" alt="next" width="34" height="33" />',
	prevBtn: '<img src="images/buttons/prev.png" alt="prev" width="34" height="33" />'
} );

btnsPosition

Type attendu : string

Valeur par défaut : "inside"

Valeurs possibles : "inside", "outside"

Le paramètre btnsPosition permet de définir la position des boutons par rapport au conteneur global du carrousel.

Exemple :

$("div").carousel( { btnsPosition: "outside" } );

nextBtnInsertFn / prevBtnInsertFn

Type attendu : function

Valeurs par défaut : false

Les paramètres nextBtnInsertFn et prevBtnInsertFn permettent d'insérer les boutons "Next" et "Previous" de manière entièrement libre.
Un paramètre peut éventuellement être défini pour la fonction. Ce paramètre sera un objet contenant tous les éléments utilisés par le script.
La fonction doit retourner l'élément associé.
Ces paramètres ont la priorité sur les paramètres nextBtn, prevBtn et btnsPosition.

Exemple :

$("div").carousel( {
	nextBtnInsertFn: function(elts) {
		return $('<button>Next</button>').prependTo("body");
	},
	prevBtnInsertFn: function(elts) {
		return $('<button>Prev</button>').prependTo("body");
	}
} );

autoSlide

Type attendu : boolean

Valeur par défaut : false

Le paramètre autoSlide permet de faire défiler autoamtiquement le carrousel à intervalle régulière, si défini sur true.

Exemple :

// Clic automatique sur le bouton "Next" (défilement automatique)
$("div").carousel( { autoSlide: true } );

autoSlideInterval

Type attendu : number

Valeur par défaut : 3000

Dépend de : autoSlide défini à true

Le paramètre autoSlideInterval permet de définir le délai (en millisecondes) entre chaque défilement, lorsque l'option autoSlide est définie à true.

Exemple :

// Clic automatique sur le bouton "Next" toutes les 5 secondes (défilement automatique)
$("div").carousel( { 
	autoSlide: true, 
	autoSlideInterval: 5000
} );

delayAutoSlide

Type attendu : number

Valeur par défaut : false (égal à autoSlideInterval)

Dépend de : autoSlide défini à true

Le paramètre delayAutoSlide permet de retarder le premier défilement, lorsque l'option autoSlide est définie à true.
Si aucune valeur n'est définie, sa valeur sera la même que celle définie pour l'option autoSlideInterval.

Exemple :

// Click automatique sur le bouton next toutes les 5sec, avec un décalage de 2sec à chaque click
$("div").carousel( { 
	autoSlide: true, 
	autoSlideInterval: 5000,
	delayAutoSlide: 2000
} );

combinedClasses

Type attendu : boolean

Valeur par défaut : false

Le paramètre combinedClasses permet d'emuler les sélecteurs de classes multiples pour Internet Explorer 6, si défini sur true.
Lorsque les classes "next" et "disabled" ou "previous" et "disabled" seront présentes, les classes "next-disabled" et "previous-disabled" seront également ajoutées.

Exemple :

$("div").carousel( { combinedClasses: true } );

effect

Type attendu : string

Valeur par défaut : "slide"

Valeurs possibles : "no", "slide", "fade"

Le paramètre effect permet de définir l'effet qui sera être utilisé lors du défilement.

Exemple :

// Carrousel en fade
$("div").carousel( { effect: "fade" } );

slideEasing

Type attendu : string

Valeur par défaut : "swing"

Valeurs possibles : "swing", "linear", valeurs apportées par jQuery Easing Plugin si présent.

Le paramètre slideEasing permet de définir le type de transition (easing) qui sera utilisé sur l'animation du carrousel.
Par défaut, jQuery propose les valeurs swing (ralentissement sur la fin de l'animation) et linear (animation linéaire).
Si le plugin jQuery Easing Plugin est présent, beaucoup d'autres valeurs sont possibles.

Exemple :

// Carrousel en slide avec un effet easing de type bounce
$("div").carousel( { slideEasing: "easeOutBounce" } );

useAddress

Type attendu : boolean

Valeur par défaut : false

Le paramètre useAddress permet d'utiliser le plugin jQuery Address. Ce plugin permet d'utiliser les ancres dans la barre d'adresse pour utiliser les fonctionnalités natives du navigateur (suivant / précédent, favoris, partage de lien).

Exemple :

// Utilisation d'une ancre dans la barre d'adresse
$("div").carousel( { useAddress: true } );
